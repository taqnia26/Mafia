### 1. Professional Playing Cards (SVG)

**Replace text cards with visual SVG playing cards:**

```typescript
// Components needed:
interface Card {
  value: number; // 1-13 (Ace-King)
  suit: 'hearts' | 'diamonds' | 'clubs' | 'spades';
}

// Card component structure:
const PlayingCard = ({ card, isHidden }: { card: Card, isHidden?: boolean }) => {
  return (
    <div className="playing-card">
      {isHidden ? (
        <CardBack />
      ) : (
        <CardFace card={card} />
      )}
    </div>
  );
};
```

**Visual requirements:**
- Professional card design with suits (♠ ♥ ♦ ♣)
- Corner values (A, 2-10, J, Q, K)
- Proper colors (red for hearts/diamonds, black for clubs/spades)
- Card back design for dealer's hidden card
- 3D card appearance with shadows
- Border radius and styling

**Example SVG card structure:**

```tsx
const CardFace = ({ card }: { card: Card }) => {
  const suitSymbols = {
    hearts: '♥',
    diamonds: '♦',
    clubs: '♣',
    spades: '♠'
  };
  
  const suitColors = {
    hearts: 'text-red-600',
    diamonds: 'text-red-600',
    clubs: 'text-black',
    spades: 'text-black'
  };
  
  const valueDisplay = {
    1: 'A',
    11: 'J',
    12: 'Q',
    13: 'K'
  }[card.value] || card.value.toString();
  
  return (
    <div className={`
      w-24 h-36 bg-white rounded-lg shadow-xl border-2 border-gray-200
      flex flex-col relative
      ${suitColors[card.suit]}
    `}>
      {/* Top left corner */}
      <div className="absolute top-1 left-1 flex flex-col items-center text-sm font-bold">
        <span>{valueDisplay}</span>
        <span className="text-xl">{suitSymbols[card.suit]}</span>
      </div>
      
      {/* Center suit */}
      <div className="flex-1 flex items-center justify-center">
        <span className="text-5xl">{suitSymbols[card.suit]}</span>
      </div>
      
      {/* Bottom right corner (rotated) */}
      <div className="absolute bottom-1 right-1 flex flex-col items-center text-sm font-bold rotate-180">
        <span>{valueDisplay}</span>
        <span className="text-xl">{suitSymbols[card.suit]}</span>
      </div>
    </div>
  );
};

const CardBack = () => {
  return (
    <div className="w-24 h-36 bg-gradient-to-br from-blue-600 to-purple-700 rounded-lg shadow-xl border-2 border-gray-200">
      <div className="w-full h-full flex items-center justify-center">
        <div className="grid grid-cols-3 gap-1 p-2">
          {[...Array(9)].map((_, i) => (
            <div key={i} className="w-3 h-3 bg-white/20 rounded-full" />
          ))}
        </div>
      </div>
    </div>
  );
};
```

### 2. Card Dealing Animations

**Smooth card animations when dealing:**

```tsx
// Animation variants
const cardVariants = {
  hidden: {
    x: -200,
    y: -100,
    scale: 0,
    rotate: -15,
    opacity: 0
  },
  visible: (custom: { delay: number, toPlayer: boolean }) => ({
    x: 0,
    y: custom.toPlayer ? 50 : 0,
    scale: 1,
    rotate: 0,
    opacity: 1,
    transition: {
      delay: custom.delay,
      duration: 0.5,
      ease: "easeOut"
    }
  }),
  flip: {
    rotateY: 180,
    transition: {
      duration: 0.6,
      ease: "easeInOut"
    }
  }
};

// Animated card component
const AnimatedCard = ({ card, delay, toPlayer }: CardProps) => {
  return (
    <motion.div
      custom={{ delay, toPlayer }}
      variants={cardVariants}
      initial="hidden"
      animate="visible"
      className="inline-block"
    >
      <PlayingCard card={card} />
    </motion.div>
  );
};
```

**Animation sequence:**
1. Cards fly from deck position (top-left)
2. Smooth arc motion to target position
3. Card flip animation when revealing dealer's hidden card
4. Stagger delay between cards (0.3s between each)
